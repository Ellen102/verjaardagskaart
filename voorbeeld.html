Verjaardagskaart

In dit project maken we een verjaardagskaart. Als je op de groene vlag klikt komen er ballonnen tevoorschijn die je kapot kan prikken door er op te klikken. Je kan het project hier vinden.
Probeer het project eens uit!

Het project klaarzetten

Je kan het lege project hier vinden. Het bevat al alle sprites die je nodig zal hebben in dit project.
Zorg ervoor dat je ingelogged bent en ga naar de project pagina hierboven. Klik op Remix zodat Scratch een kopie van het project op jouw naam maakt en alle verandering worden dan opgeslagen. Kijk eens welke sprites er allemaal aanwezig zijn in het project. Je mag de tekst aanpassen.

De ballon laten stijgen

Om de ballon te laten stijgen moeten we de y-waarde van de ballon aanpassen. Herinner je je de x- en y-as uit de wiskunde les? Het is mooier als we de ballon in een aantal kleinere stappen naar boven laten gaan in plaats van in één grote stap. Iedere keer als we op de groene vlag klikken, moet de ballon terug onderaan starten.
Programmeer de ballon zodat deze van onder tot aan het plafon (paarse lijn) beweegt elke keer we op de groene vlag klikken.
Jouw code zou op de code hieronder moeten lijken. Kies zelf de getallen!

Laat de ballon ronddraaien

Het is natuurlijk leuker als de ballon ook ronddraait.
Voeg de onderstaande code toe aan het project. Vul zelf getallen in en kijk wat er gebeurt. De andere code laat je staan.
when gf clicked repeat {} turn cw {} degrees end
Welke waarden moeten we invullen zodat de ballon volledig zou ronddraaien? Scratch kan dit voor ons berekenen! We weten dat een cirkel 360 graden is en jij kiest in hoeveel stappen je dit willen doen. Hoe meer stappen hoe trager de ballon draait. Dus voeren we een berekening uit
({360}/{})
.
Als we de ballon nu sneller willen laten draaien, moeten we dit op twee plaatsen aanpassen. Een keer in het herhaal-block en één keer in de deling. Je kan gemakkelijk één van de twee vergeten aanpassen... Gelukkig is hier een oplossing voor! We gebruiken hiervoor een variabele, bijvoorbeeld
(i)
. Je kan een variable maken in de toolbox bij de categorie Variable.
Pas je code aan zodat je een variable gebruikt om de snelheid waarmee de ballon draait aan te geven.
Je code ziet er nu als volgt uit:
when gf clicked set [i] to {} repeat (i) turn cw ({360}/(i)) degrees end

Een echte ballon is een beetje doorzichtig

Je kan de ballon een beetje doorzichtig maken door gebruik te maken van het geest effect, zo zorg je ervoor dat je het plafon (de paarse lijn) en de letters er door kan zien.
set [ghost] effect to {}
Maak de ballon een beetje doorzichtig zodat je de tekst en het plafon erdoor ziet.

Een paar verassingen...

Als je bij uiterlijken gaat kijken zie je dat de ballon drie verschillende kleuren kan hebben. We kunnen hier zelf eentje van kiezen.
switch costume to [balloon1-b]
Of we kunnen er willekeurig eentje door de computer laten kiezen.
switch costume to (pick random {1} to {3})
Pas je programma aan zodat iedere keer je op de groene vlag drukt een willekeurige uiterlijk voor de ballon wordt gekozen.
Je kan nog meer willekeurig maken. Bijvoorbeeld de start positie van de ballon (x-waarde) of hoe snel de ballon ronddraait. Misschien denk jij nog aan andere dingen?
Pas je programma aan zodat de ballon steeds op een andere locatie start wanneer je op de groene vlag klikt.
Zorg ervoor dat de ballon steeds met een andere snelheid ronddraait.

Meer ballonnen

Weet je wat een Kloon is? Dit een kopie dat er hetzelfde uitziet en zich hetzelfde gedraagd als het orgineel. In Scratch kan je ook klonen!
Voeg het volgende stukje code toe aan je project. Wat gebeurt er?
when gf clicked create clone of [mezelf]
Er zijn nu twee ballonnen maar de andere ballon doet nog niets. Er zijn verschillende manieren om dit probleem op te lossen, we bespreken hier één mogelijke oplossing.
Vervang op de andere plaatsen (dus niet in de code hierboven)
when gf clicked
door
when I start as a clone
. Wat is er verandert?
Als je nu op de groene vlag klikt, lijkt het alsof hetzelfde gebeurt, maar het is nu de kloon die beweegt. Als we nu twee ballonnen willen die bewegen, kunnen we nog een kloon maken. Maar wat dan met het origineel? We verbergen het gewoon!
Zorg ervoor dat er twee ballonen zijn die stijgen en ronddraaien. Doe dit door twee klonen te maken en het origineel te verbergen.
Je code zou er ongeveer zo moeten uitzien:
when gf clicked hide repeat {2} create clone of [mezelf] when i start as a clone show
Maak zoveel ballonnen als je zelf wil!

Ballonnen kapot prikken

Zoveel ballonnen wil je natuurlijk kapot prikken. Je kan een kloon verwijderen met
delete this clone
Pas je programma aan zodat de ballonnen verdwijnen als je erop klikt. Zorg voor een bijpassend geluid.
Dit ziet er dan als volgt uit:
when this sprite clicked start sound [Pop] delete this clone
